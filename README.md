# shadowrunwiki
Wiki de Shadowrun para checagem rápida de conceitos e itens.

Este projeto é um presente e contém traduções e anotações feitas por uma
colaboradora com base na propriedade intelectual de Shadowrun, pertencente
à Catalyst Game Labs / The Topps Company. Trata-se de um projeto de fã,
sem fins comerciais, e será removido mediante solicitação dos detentores
dos direitos. Nenhum direito autoral é reivindicado sobre o conteúdo
original de Shadowrun.
